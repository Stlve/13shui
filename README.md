### 运行环境

安卓手机

### 编译与运行

java（基于AS），运行后打开虚拟机即可开始操作

### 使用方法

- 输入用户名和密码，点击登录后进入游戏主界面
- 点击“Enter”，进入游戏
- 游戏界面出现得到的牌和出牌结果，点击返回可“返回”游戏主界面
- 点击“玩法说明”，出现简要玩法说明
- 点击“往期对战情况”，出现每次战局中自己的牌型和分数，可点击继续查看详情
- 点击“排行榜”，可出现所有玩家的实时排名
- 点击“返回”，可返回登录界面
